# React Landingpage mit Shadcn UI

Eine moderne Landingpage erstellt mit React, TypeScript, Vite und Shadcn UI.

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

## Build

```bash
npm run build
```
